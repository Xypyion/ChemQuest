# คู่มือการใช้งานสำหรับนักเรียน (Student Guide)

ยินดีต้อนรับสู่ **ChemQuest** การผจญภัยเรียนรู้วิชาเคมี! 🧪

## เริ่มต้นใช้งาน
1. เปิดเว็บไซต์ที่ `http://localhost:4000`
2. กดแท็บ **สมัครสมาชิก** แล้วกรอกชื่อ อีเมล และรหัสผ่าน (อย่างน้อย ๖ ตัวอักษร)
3. เลือกระดับความยากที่ต้องการ: **ง่าย / ปานกลาง / ยาก**
4. กด **สร้างนักสำรวจ** เพื่อเริ่มเล่น
5. สลับภาษาไทย/อังกฤษได้จากปุ่ม **EN | ไทย** ที่มุมขวาบน

## แผนที่การผจญภัย 🗺️
- คุณจะเดินทางผ่าน ๓ ภูมิภาค: **ทุ่งหญ้า → หุบเขาถ่านไฟ → ยอดฟ้า**
- แต่ละด่านจะปลดล็อกทีละด่าน ด่านที่เล่นได้จะมีตัวการ์ตูนโบกมือ พร้อมป้าย “แตะเพื่อเล่น!”
- ด่านที่ล็อกอยู่จะมีสัญลักษณ์ 🔒 (แตะเพื่อดูเหตุผลว่าทำไมยังเข้าไม่ได้)

## เมื่อแตะที่ด่าน — กระดานประจำด่าน
จะพบ ๓ เมนู
1. **📖 เริ่มบทเรียน** — อ่านเนื้อเรื่องจากรูบี้ ดูวิดีโอ แล้วทำ **แบบทดสอบก่อนเรียน**
2. **📒 ส่งงาน** — โพสต์ผลงาน ดูงานของเพื่อน และพูดคุยกัน
3. **🧾 แบบทดสอบหลังเรียน** — จะเปิดให้ทำเมื่อคุณครูเปิด

## การเรียนและการทำแบบทดสอบ
- อ่านเนื้อเรื่องของรูบี้ทีละหน้า กด **ถัดไป ▶** หรือ **ข้ามเนื้อเรื่อง**
- ดูวิดีโอที่แทรกอยู่ในเนื้อเรื่อง
- ทำแบบทดสอบ: แตะเลือกคำตอบ ระบบจะบอกทันทีว่าถูกหรือผิด พร้อมคำอธิบาย
- หากมี **ตัวจับเวลา** ให้รีบทำให้ทัน เมื่อหมดเวลา ระบบจะบันทึกคำตอบล่าสุดให้
- ทำคะแนนได้ตั้งแต่ **๖๐%** ขึ้นไป จะ **ผ่าน** และได้รับ **เกียรติบัตร** 🎖️

## การปลดล็อกด่านถัดไป
- การผ่าน **แบบทดสอบก่อนเรียน** จะได้เกียรติบัตร แต่ยังไม่ปลดล็อกด่านถัดไป
- ต้อง **ผ่านแบบทดสอบหลังเรียน** (ที่คุณครูเปิดให้) จึงจะปลดล็อกด่านถัดไป

## กระดานส่งงาน 📒
- โพสต์ผลงานของคุณ พร้อมแนบไฟล์ได้: รูปภาพ, PDF หรือไฟล์ใดก็ได้ (สูงสุด ๖ ไฟล์ ไฟล์ละ ๘ MB)
- รูปภาพจะแสดงตัวอย่างให้เห็นทันที
- กด **❤️ ถูกใจ** และ **แสดงความคิดเห็น** ผลงานของเพื่อนได้
- ใต้โพสต์ **งานที่มอบหมายของคุณครู** จะมีช่อง **“นักเรียนสงสัยอะไรมั้ย”**
  ใช้ถามคุณครูแบบส่วนตัว — **มีเพียงคุณครูเท่านั้นที่เห็นข้อความนี้**

## รางวัลและอันดับ
- **🎖️ เกียรติบัตร** — สะสมได้จากทุกด่านที่ผ่าน ดูได้ที่เมนูเกียรติบัตร
- **🏆 กระดานคะแนน** — เก็บคะแนนจากการทำแบบทดสอบเพื่อไต่อันดับกับเพื่อน ๆ

ขอให้สนุกกับการผจญภัยเรียนรู้เคมี! 🚀
